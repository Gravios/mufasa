┌──────────────────────────────────────────────────────────────────────────────┐
│  KEY CROSS-CLUSTER RELATIONSHIPS  (the load-bearing arrows)                  │
└──────────────────────────────────────────────────────────────────────────────┘

  Workbench startup
  -----------------
  ui_qt/workbench_app.py
    → ui_qt/qt_confirm.py    (installs Qt confirm override; patch 122cj)
    → ui_qt/recent_project.py
    → ui_qt/operations.py    (build sidebar)
    → ui_qt/pages/*          (mount per-workflow pages)

  Pages → Forms (one-to-many)
  ---------------------------
  ui_qt/pages/X_page.py
    → ui_qt/forms/X_form_1.py    \
    → ui_qt/forms/X_form_2.py     >  each form is one operation
    → ui_qt/forms/X_form_3.py    /

  Form → Backend (collect args → run_with_progress → backend.target)
  -----------------------------------------------------------------
  ui_qt/forms/X.py
    .collect_args()
    .target(config_path=…, kwargs=…)
      → mufasa.project_layout.project_paths_from_config(config_path)
      → mufasa.mixins.config_reader.ConfigReader(config_path=…)
      → <backend cluster>.<class>(...).run()

  Per-cluster backend hop (sample chain — outlier correction)
  -----------------------------------------------------------
  ui_qt/forms/pose_cleanup.py
    → outlier_tools/outlier_corrector_movement.py
       → mixins/config_reader.py   (resolve dirs)
       → mixins/feature_extraction_mixin.py (distance helpers)
       → utils/checks.py           (validation)

  Visualization route resolution (122dc)
  --------------------------------------
  ui_qt/forms/visualizations.py
    .target(route, config_path, …)
      → _resolve_viz_source_dir(config_path, project_root, source_name)
         → _VIZ_SOURCE_V1_MAP[source_name]   (v1 path)
         → or <project_root>/csv/<source_name>/   (legacy path)
      → plotting/<backend>.py.<class>(data_paths=…).run()

  ROI Apply-all v1 path (122d9 / 122da fixes)
  -------------------------------------------
  ui_qt/dialogs/roi_video_table.py
    → roi_tools/roi_utils.py
       multiply_ROIs() / reset_video_ROIs()
         → project_layout.project_paths_from_config(config_path)
            ["video_dir", "roi_definitions_path"]

  Frame extraction v1 path (122db fix)
  ------------------------------------
  ui_qt/forms/video_processing.py
    → video_processors/video_processing.py
       extract_frames_from_all_videos_in_directory(config_path, …)
         → project_paths_from_config(config_path)["project_root"]
         → branch by .endswith(".toml") →
              v1:     <root>/derived/frames/extracted/<video>/
              legacy: <root>/frames/input/<video>/

  Migration tool
  --------------
  cli/migrate_project.py  (python -m invocation; not console-script)
    → legacy_layout.LEGACY_TO_V1_MAPPING (the move table)
    → project_layout.write_project_toml (emit the v1 project.toml)
    → project_layout.detect_layout (idempotence guard)
